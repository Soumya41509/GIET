# 🎓 Team Nexus — Student Grievance App

> A dedicated student-side grievance and feedback application built with React Native & Expo. Designed to simplify complaint submission, improve transparency, and enable seamless communication with administration.

---

## 🧭 Overview

The **Team Nexus Student Grievance App** is a mobile-first solution that empowers students to easily submit, track, and manage their grievances. It ensures a smooth and transparent experience while interacting with the institution’s grievance redressal system.

---

## 🚀 Features

* 🧾 **Quick Grievance Submission** – Raise complaints with proper details, images, and categories
* 🔔 **Live Status Tracking** – Real-time updates on grievance progress
* 📂 **Category-Based Filing** – Structured complaint system for better handling
* 🔁 **Reopen Grievance** – Reopen resolved issues with valid reason & proof
* 📨 **Smart Notifications** – Instant alerts for updates and actions
* 🧠 **User-Friendly UI** – Clean and intuitive interface for students
* 🔐 **Secure Login System** – Authentication with protected user data

## 🧾 Usage Flow

1. Student logs in using credentials
2. Submits grievance with category and details
3. Tracks status from dashboard
4. Receives notifications on updates
5. Can reopen grievance if not satisfied
