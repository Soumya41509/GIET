# 🛠️ Team Nexus — Admin Panel (Grievance Management System)

> A powerful admin-side control panel built to efficiently manage, monitor, and resolve student grievances. Designed for structured workflow, transparency, and faster resolution across departments.

---

## 🧭 Overview

The **Team Nexus Admin Panel** is the core management system of the grievance redressal platform. It enables administrators to oversee all grievances, assign cases, track progress, and maintain complete control over the system.

---

## ⚙️ Tech Stack

| Category            | Technology              |
| ------------------- | ----------------------- |
| **Frontend**        | React (Web Admin Panel) |
| **Backend**         | Supabase                |
| **Database**        | Supabase PostgreSQL     |
| **Hosting**         | Web Deployment          |
| **Version Control** | Git & GitHub            |

---

## 🧾 Usage Flow

1. Admin logs into the panel
2. Views all incoming grievances
3. Assigns cases to concerned staff
4. Monitors progress and updates status
5. Handles reopened cases and verifies proof
6. Manages categories, FAQs, and support queries

## 🧠 Project Goals
