# 👨‍💼 Team Nexus — Staff App (Grievance Handling System)

> A dedicated staff-side mobile application designed to manage, respond to, and resolve assigned grievances efficiently. Built for smooth coordination between administration and ground-level staff.

---

## 🧭 Overview

The **Team Nexus Staff App** empowers staff members to handle grievances assigned to them in a structured and efficient manner. It ensures faster resolution, clear communication, and accountability at every stage.

---

## ⚙️ Tech Stack

| Category            | Technology          |
| ------------------- | ------------------- |
| **Frontend**        | React Native (Expo) |
| **Backend**         | Supabase            |
| **Database**        | Supabase PostgreSQL |
| **Version Control** | Git & GitHub        |

---

## 🧾 Usage Flow

1. Staff logs into the app
2. Views assigned grievances
3. Updates status and adds remarks
4. Uploads proof if required
5. Marks grievance as resolved

---

## 🧠 Project Goals

* Enable faster grievance resolution at staff level
* Improve coordination between admin and staff
* Ensure accountability with proper tracking
* Provide a simple and efficient workflow
